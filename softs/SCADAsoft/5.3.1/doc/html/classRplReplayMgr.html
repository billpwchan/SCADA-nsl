Member Enumeration Documentation

enum RplReplayMgr::WorkingState
Description:
enum replay manager state (see getInfos() function). 
      enum WorkingState {
         WAITFORINIT = 1,
         READY,
         RUNNING,
         FREEZE
      };
Values:
  • WAITFORINIT : the replay manager is waiting for initialization.
  • READY : the replay manager is ready and is waiting for start.
  • RUNNING : the replay manager is running.
  • FREEZE : the replay manager is freezed.
