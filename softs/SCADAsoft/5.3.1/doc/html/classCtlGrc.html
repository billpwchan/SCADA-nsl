enum CtlGrc::ScsCtlGrcMode
Description:
launching mode value list. 
      enum E_ScsCtlGrcMode { 
         Auto = 1,
         StopOnFailed,
         StopOnFirstBRCExecuted
      } ScsCtlGrcMode;
Values:
  • Auto: The GRC is excuted sequentially,
  • StopOnFailed: The GRC execution stopped when a step execution failed.
  • StopOnFirstBRCExecuted: The GRC execution stopped when the first step is executed (Not necessary the BRC with index=1). This allows to execute only one BRC of a GRC
enum CtlGrc::ScsCtlGrcStatus
Description:
enum corresponding to the possible values for the current status of a GRC. 
      enum E_ScsCtlGrcStatus { 
         Terminated = 1,
         WaitForRun,
         Initializing,
         Running,
         Waiting,
         Stopped,
         Aborted,
         Suspended,
         Resuming
      } ScsCtlGrcStatus;
Values:
  • Terminated: Execution terminated,
  • WaitForRun: Waiting for the GRC to be run,
  • Initializing: Initializing execution,
  • Running: Execution running,
  • Waiting: Execution waiting,
  • Stopped: Execution stopped,
  • Aborted: Execution aborted.
  • Suspended: Execution suspended.
  • Resuming: resuming in progress.
