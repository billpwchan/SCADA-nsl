typedef void(* AgtSupervisionCallback)(const char *processName, const char *state, void *arg)

Prototype of the AscAgent notification callback function. 

 #include "agent.h"
Description:
this function is a client application function called by the AscAgent object to asynchronously notify a client subscribed to the state of a process in a current physical environment.
Parameters:
processName[in]: name of the process,
state[in]: current state of the process,
arg[in]: user optional argument given by the client when it calls the subscription function and returned to it when the current callback function is called by the AscAgent. This parameter is optional, but, if set, could be used to give back information to the client.
Return values:
none.
