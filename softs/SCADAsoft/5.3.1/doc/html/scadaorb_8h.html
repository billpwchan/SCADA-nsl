template<typename SMART_PTR , class SERVANT_CLS >
void activate ( SMART_PTR &  ptr,
SERVANT_CLS *  servant 
)
Description:
this function initializes a CORBA smart pointer (either a CORBA servant ptr or var) with a SOAP CORBA servant instance and activates the servant. When this method returns, the servant is then able to process incoming CORBA request.

Notice that the servant must have been created in the heap.

Parameters:
ptr[out]: a smart pointer to be initialized with the servant address.
servant[in]: a heap allocated SOAP CORBA servant that shall be activated.
