Detailed Description

This class provides a remote access to OlsList objects.

This class is used to remotely access the services of OlsList objects managed by a distant server process. It inherits from the methods of the OlsServer class and has an additional list modification method. The methods of this class are exactly the same than thoses of OlsList class, except than they have in first parameter the name of the list.

const char *  name = "OlsManager",
ScsPriorityLevels  priority = SCS_PRIORITY_NORMAL 
)
Description:
constructor. This function creates an interface object for remote OlsList objects. It creates a connection to the server which manages the remote list objects. This server is identified by the logicalEnv and serverName parameters. For example, to access the services of an alarm list managed by an alarm server "AlmServer" running in the physical environment "ENV1_cactus", call the current constructor like this : 
 OlsManager *olsManager = new OlsManager("ENV1", "AlmServer");
Parameters:
environment[in]: name of the logical SCADAsoft environment in which the server managing the remote OlsList objects is running,
name[in]: name of the server which manages the OlsList objects.
priority[in]: priority of the messages to the OlsManager (only useful with redudancy).
Return values:
none. The result of the connection can be obtain by calling the getStatus() function.

Description:
this function modifies the remote list according to the data contained in the dataset.
Parameters:
listName[in]: name of the remote list,
dataSet[in]: dataset containing the data to add, modify or delete.
Returns:
  • ScsValid : success.
  • OlsBadListNameError : the remote server does not export the specified list.
  • OlsBadDataKeyError : if one key in the dataset is not does not correspond to a data of the list.
  • OlsDataModeUndefinedError : if one data in the dataset has an OlsData::UNKNOWN type.
  • OlsErrHostServer : cannot connect to the ascmanager process.
  • OlsORBError : communication failure with the remote server.
