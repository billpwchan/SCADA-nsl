enum ScsPriorityLevels 
 #include "scsdefs.h"
Description:
this enum represents the possible priority levels for the messages sent to a server running in a redunded (either local or remote) SCADAsoft environment. The priority is set globally using a parameter of the constructor of the server interface class. The possible values are the following : 
   enum ScsPriorityLevels {
     SCS_PRIORITY_NORMAL = 0,
     SCS_PRIORITY_HIGH,
     SCS_PRIORITY_LOW,
     SCS_PRIORITY_URG,
     SCS_PRIORITY_LOWER,
     SCS_PRIORITY_LOWEST
   };
Values:
  • SCS_PRIORITY_LOW: low priority level,
  • SCS_PRIORITY_NORMAL: default priority level,
  • SCS_PRIORITY_HIGH: high priority level,
  • SCS_PRIORITY_URG: most urgent priority level,
  • SCS_PRIORITY_LOWER: lower priority level,
  • SCS_PRIORITY_LOWEST: lowest priority level.
