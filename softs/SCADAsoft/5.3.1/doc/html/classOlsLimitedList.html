Detailed Description

This class is one of the class of the OLS (Object List Server) SCADAsoft component.

This class is used to store data elements. An object (object of OlsData class) represents a set of basic data (object of OlsField class). An object list (object of OlsList class) groups a set of objects containing the same kind of information. The OlsLimitedList class behaves as OlsList except the number of OlsData within the list is limited. If the limit is reached, the oldest data is removed from the list and every client are notified.
