Detailed Description

This class is the main class of SCADAsoft.

The class Scadasoft contains general functions relative to the SCADAsoft initialization, the mainloop, the management of traces, the info concerning the time, the current process or the redundancy. 

 #include "scs.h"

Member Enumeration Documentation

enum Scadasoft::RedundancyType
Description:
enum corresponding to the possible values of redundancy type. 
     enum RedundancyType { 
       SCS_NO_REDUNDANCY = 0,
       SCS_SYNCHRONOUS_REDUNDANCY,
       SCS_ASYNCHRONOUS_REDUNDANCY,
       SCS_COLD_REDUNDANCY
     };
Values:
  • SCS_NO_REDUNDANCY: no redundancy,
  • SCS_SYNCHRONOUS_REDUNDANCY: synchronous redundancy (default type for redundancy),
  • SCS_ASYNCHRONOUS_REDUNDANCY: asynchronous redundancy (one online server and one standby server running simultaneously, but no synchronisation using InputManager),
  • SCS_COLD_REDUNDANCY: cold redundancy (only one online server running).

Description:
this method is the SCADAsoft initialization method. It must be called at the beginning of each program using the SCADAsoft library, before calling any other SCADAsoft function. It initializes the server name and environment used by any process in every communication with the current process. Returns:

static ScsStatus Scadasoft::SetUserTrace ( int  moduleMask,
int  levelMask 
) [static]
Description:
this method allows to activate up the traces for user modules and levels. The moduleMask parameter is a 32 bits mask defining for which user module(s) the traces are activated (each user component is associated with a module number) and levelMask is the 32 bits mask defining the level(s) for which the traces are activated. To produce a trace using SCADAsoft, a macro should be defined for each component in a common include file. For example, for the Ccc component : 
 #define CccTrace ScsTraceFunction(module)
where module is the module number (between 0 and 31) associated with the component. This CccTrace macro is called in the source code in the following way : 
 CccTrace(SCS_LEVEL(level), format [, arguments]);
where level is the level of the trace (between 0 and 31), format is the output format string (as for the printf() C standard library function) and arguments optional arguments corresponding to the output format string. For example : 
 CccTrace(SCS_LEVEL(3), "Activation of timer %d(%s)", timerId, timerStr);
Parameters:
moduleMask[in]: user module mask,
levelMask[in]: level mask.
Returns:
  • ScsValid : success.
