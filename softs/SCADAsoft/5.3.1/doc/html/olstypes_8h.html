typedef void(* OlsCallback)(const char *listName, int id, const OlsDataSet &dataSet, void *arg) 
 #include "olstypes.h"

Prototype of the function which is called by an Ols server to notify a client subscribed on a list (see OlsServer::subscribe()), about the changes which occurred on this list.

Parameters:
listName[in]: name of the list,
id[in]: subscription id,
dataSet[in]: dataset containing the data created, modified or deleted,
arg[in]: user optional argument passed as parameter of the callback when called.
Return values:
none.

Enumeration Type Documentation

enum OlsNotificationMode 
 #include "olstypes.h"
Description:
enum corresponding to the notification mode used by an Ols server to notify a subscriber which is inside its environment (see OlsServer::subscribe()). (an Ols server notify asynchronously subscribers which are outside its environment). 
    enum  OlsNotificationMode {
      OLS_DEFAULT_NOTIFY = 0,
      OLS_SYNCHRONOUS_NOTIFY,
      OLS_ASYNCHRONOUS_NOTIFY
    };
Values:
  • OLS_DEFAULT_NOTIFY: If the environment variable SCSSYNCNOTIFY is set, the Ols server will notify the local subscriber synchrounously, Otherwise the Ols server will notify the local subscriber asynchronously.
  • OLS_SYNCHRONOUS_NOTIFY: The Ols server will notify the local subscriber synchrounously.
  • OLS_ASYNCHRONOUS_NOTIFY: The Ols server will notify the local subscriber asynchronously.
