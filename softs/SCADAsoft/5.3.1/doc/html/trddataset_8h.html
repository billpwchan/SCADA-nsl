typedef void(* TrdUpdateCallback)(TrdDataSetList &dataSetList, void *arg) 
 #include "trddataset.h"

Prototype of the TRD component notification callback. The dataSetList is a list of datasets, each one of which is associated with an attribute and contains the values from the creation of the archive containing the attribute. Each data of a dataset can be retrieved and removed from the dataset using the TrdDataSet::popData () function.

Parameters:
dataSet[in]: dataset containing the list of the attribute values,
arg[in]: user optional argument passed as parameter of the subscribe() method.
Return values:
none.
