enum TscTaskType 
 #include "tsctypes.h"
Description:
enum corresponding to the task type. 
    enum TscTaskType {
    TSC_TASKTYPE_EXEC = 0,
    TSC_TASKTYPE_CMD,
    TSC_TASKTYPE_GRC,
    TSC_TASKTYPE_EVT,
    TSC_NB_TASKTYPE
    };
Values:
  • TSC_TASKTYPE_EXEC: the task execute a command in background,
  • TSC_TASKTYPE_CMD: the task call CtlCmdServer::sendMCommand in the local environment,
  • TSC_TASKTYPE_GRC: the task call CtlGrcServer::launchGrc in the local environment,
  • TSC_TASKTYPE_EVT: the task send an event to a process in the local environment, the process will execute the function registered by AscServer::registerEventAscCallback.
