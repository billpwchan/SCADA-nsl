Detailed Description

Interface class for Calculation Engine functions.

This class is the interface class for the services exported by the DbmServer for the implementation of Calculation Engine functions.

void DbkSession::endOfUpdate ( )
Description:
Updates the alarm and list servers if needed. That method may be called after several setValue to trigger the dependent servers of the environments. It has the same effect than the endCECycle at the end of a write. That method must not be used inside a CE function. It must be used after the endWrite.

state[in]: alarm state ("A" for alarm, "N" for normal or "T" for transitory),
valueAutomatonName[in]: name of the value automaton,
ackAutomatonName[in]: name of the ack automaton,
equipmentDate[in]: equipement date (applicative value)
acquisitionDate[in]: acquisition date (applicative value)
severity[in]: alarm severity,
functionalCat[in]: functional category (applicative value), used for alarm synthesis,
geographicalCat[in]: geographical category (applicative value), used for alarm synthesis,
equipmentName[in]: equipment name,
message[in]: textual description of the alarm,
group1[in]: group number one (optional parameter, its default value is ""),
group2[in]: group number two (optional parameter, its default value is ""),
thresholdLow[in]: low threshold of the range, should be a negative value if used (optional parameter, its default value is 0),
thresholdHigh[in]: high threshold of the range, should be a negative value if used (optional parameter, its default value is 0).
