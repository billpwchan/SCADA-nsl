Detailed Description

This class is one of the class of the OLS (Object List Server) SCADAsoft component.

This class is used to filter the data in case of reading or subscribing request on a data list. For each such request, the client can specify a filter. A filter is composed of a field list and contains also a criteria (see OlsCriteria, OlsAndCriteria, OlsOrCriteria or OlsFieldCriteria classes). A criteria is a logical combination (NOT, AND or OR) of elementary conditions than the data must fulfill for it to be part of the request result. An elementary condition is made of a field name, an operation and a comparison value.
