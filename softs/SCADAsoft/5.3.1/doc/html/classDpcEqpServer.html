Detailed Description

This class is the interface class for the exported services of the DPC component (Data Processing and Calculation): it allows to change an equipment status or an internal variable status.

This class is the interface class for the services exported by the DPCEQP server. It is used to change an equipment status or an internal input variable (IIV) status. 

 #include "dpc.h"

enum DpcEqpServer::ForcedStatus
Description:
enum corresponding to the possible values for the new forced status to be given to an internal input variable (see changeVarForce() functions). 
      enum ForcedStatus { 
         NOT_FORCED = 0,
         FORCED
      };
Values:
  • NOT_FORCED : the value of the internal input variable takes back the field value and the state "Valid - Forced value" is annulated for the status,
  • FORCED : the value of the internal input variable is forced and the status is changed to "Valid - Forced value", only if the variable is valid.

enum DpcEqpServer::ValidityStatus
Description:
enum corresponding to the possible values for the new validity status to be given to an internal input or output variable (see changeVarStatus() function). 
      enum ValidityStatus { 
         OPERATOR_INHIBIT = 0,
         VALID,
         ALARM_INHIBIT_VAR,
         NO_ALARM_INHIBIT_VAR,
         CHATTER_INHIBIT_VAR,
         NO_CHATTER_INHIBIT_VAR
      };
Values:
  • OPERATOR_INHIBIT : the variable is invalided with state "Invalid - Variable inhibited",
  • VALID : this value suppress the states of invalidity "Invalid - Variable inhibited".
  • ALARM_INHIBIT_VAR : an alarm inhibition is set on the variable (the operator invalidation state is not changed),
  • NO_ALARM_INHIBIT_VAR : the alarm inhibition is removed (the operator invalidation state is not changed).
  • CHATTER_INHIBIT_VAR : the variable is invalided due to IO chattering detection.
  • NO_CHATTER_INHIBIT_VAR : the chattering detection is off, suppress the inhibition due to chattering.
enum DpcEqpServer::WorkingStatus
Description:
enum corresponding to the possible values for the new working status to be given to an equipment (see changeEqpStatus() function). 
      enum WorkingStatus { 
         OUT_OF_SERVICE = 0,
         CONTROL_INHIBITED,
         IN_SERVICE,
         ALARM_INHIBIT_EQP,
         NO_ALARM_INHIBIT_EQP
      };
Values:
  • OUT_OF_SERVICE : the working status of the equipment is "out of service" (full inhibited). All the monitoring and control functions are inhibited.
  • CONTROL_INHIBITED : the working status of the equipment is "control inhibited". Control functions are inhibited, but monitoring functions remains unaffected.
  • IN_SERVICE : the working status of the equipment is "in service". All the monitoring and control functions are normal.
  • ALARM_INHIBIT_EQP : the working status of the equipment is "valid - alarm inhibit". An alarm inhibition is set on the equipment. The monitoring and control functions are not changed.
  • NO_ALARM_INHIBIT_EQP : the alarm inhibition is removed on the equipment. The monitoring and control functions are not changed.

WorkingStatus  newStatus,

)
Description:
this function changes the working status of an equipment. IN_SERVICE: suppress for the IIV and IOV variables the state of invalidity "Invalid - Equipment inhibited"; no effect on the other states of invalidity. CONTROL_INHIBITED: invalidate only the IOV variables with the state "Invalid - Equipment inhibited". OUT_OF_SERVICE: invalidate all the IOV and IIV variables with the state "Invalid - Equipment inhibited".
Parameters:
equipmentName[in]: name of the equipment (Rtap alias or symbolic address).
newStatus[in]: new working status to be assigned.
source[in]: source of the request. This parameter is used for event logging: when it is not set, no event is logged when this function is called.
Returns:
  • ScsValid : success.
  • DpcDBAccessError : database access error.
  • DpcEquipmentStatusError : equipment status error.
  • DpcORBError : ORB communication error (exception raised).
  • DpcScadasoftFunctionError : SCADAsoft function error.
