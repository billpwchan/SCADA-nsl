Detailed Description

This class is one of the class of the OLS (Object List Server) SCADAsoft component.

This class is used to store data elements. An object (object of OlsData class) represents a set of basic data (object of OlsField class). An object list (object of OlsListWithClientVisibility class) groups a set of objects containing the same kind of information. An object of OlsListWithClientVisibility class contains a dataset containing the fields of the elements of the list, a list of filters, a list of subscribers and the data elements list. A Visbility field of type ScsINTEGER is automatically added to the list. It maintains wether or not a particuler OlsData has been sent to a client. It is restricted to 32 clients.
