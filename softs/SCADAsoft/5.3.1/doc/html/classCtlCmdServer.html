Detailed Description

This class allows to request the execution of commands list.

This class is the interface class for the services exported by the CTLCMD server. It is used to request the execution of commands list. 

 #include "ctlcmd.h"

ScsStatus CtlCmdServer::sendCommand ( int  nbCommands,
CtlCmdCommand *  commandsList,
const char *  source = NULL,
void *  userArg = NULL 
)
Description:
this function sends a command (IOV, internal output variable) or a list of commands to execute to the DAC component. When several commands are specified into the list, this means that they should be treated together. They are all sent to DAC or no one: if one of the commands of the list can not be sent (no valid command, initial condition not correct, ...), no command is sent and the current function returns with an error status corresponding to the error. The initial conditions of all the commands of the list are verified, but only the return condition of the first command is verified. Warning: individual commands should not be sent into a list.
Parameters:
nbCommands[in]: number of commands.
commandsList[in]: list of commands.
source[in]: source of the request (7 characters max). This parameter is used for event logging: when it is not set, no event is logged when this function is called.
userArg[in]: optionnal user argument. This argument is given back when the command return callback is called by the CTLCMD server.
Returns:
  • ScsValid : success.
  • CtlCommandSendingError : error sending a command to the DAC component.
  • CtlDBAccessError : database access error.
  • CtlDBMError : an error was returned by a function of the DBM component.
  • CtlExecutionError : command variable already executing.
  • CtlInternalError : internal error.
  • CtlMemoryError : memory allocation error.
  • CtlORBError : ORB communication error (exception raised).
  • CtlTimerError : timer activation error.
  • CtlVariableNameError : variable identification error (variable not in DB or ambiguous identification).
  • CtlVariableStatusError : variable status error (not valid).
  • CtlVariableTypeError : variable type error.
  • CtlVariableValueError : bad value to be given to the variable.
