Detailed Description

This class allows queries on data from the historic past on elements of archives for specified time periods.

It is the interface class for the services exported by the HIS server providing queries on historic past values. It is used to get the list of disc files containing the subset of elements of an archive corresponding to a time period either still in memory, or saved on disc, using a filter on the attributes associated with the archive.
