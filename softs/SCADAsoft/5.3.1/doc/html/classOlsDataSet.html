Detailed Description

This class is one of the class of the OLS (Object List Server) SCADAsoft component.

This class is used to store a value into an element of a list. An object (object of OlsData class) represents a set of basic data (object of OlsField class). An object list (object of OlsList class) groups a set of objects containing the same kind of information. An object of OlsDataSet class contains a set of OlsField objects.
