Detailed Description

This class is the interface class for the exported services of the ASC SCADAsoft component (Application Supervisory and Control).

This class is used for the supervision of the processes of the current environment, when no redundancy is used. 

 #include "asc.h"

ScsStatus AscServer::setTrace ( const char *  physicalEnv,
const char *  processName,
int  moduleMask,
int  levelMask 
)
Description:
this function allows to activate up the traces for SCADAsoft modules and levels of the remote processName application running in the physicalEnv physical environment. The moduleMask parameter is a 32 bits mask defining for which SCADAsoft module(s) the traces are activated (each SCADAsoft component is associated with a module number) and levelMask is the 32 bits mask defining the level(s) for which the traces are activated.
Parameters:
physicalEnv[in]: physical environment name,
processName[in]: process name,
moduleMask[in]: SCADAsoft module mask,
levelMask[in]: trace level mask.
Returns:
  • ScsValid : success.
  • AscUnknownEnvironmentError : the physicalEnv environment is unknown,
  • AscInactiveServerError : the server is not active,
  • AscBadServerNameError : the server is unknown (no environment specified).
  • AscORBError : error in orb communication.
