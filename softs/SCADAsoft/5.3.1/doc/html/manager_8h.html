typedef void(* MgrSupervisionCallback)(const char *logicalEnv, const char *processName, const char *state, void *arg)

Prototype of the AscManager notification callback function. 

 #include "manager.h"
Description:
this function is a client application function called by the AscManager object to asynchronously notify a client subscribed to the state of a process in a logical environment (current or remote environment).
Parameters:
logicalEnv[in]: name of the logical environment in which the process is running,
processName[in]: name of the process,
state[in]: current state of the process,
arg[in]: user optional argument given by the client when it calls the subscription function and returned to it when the current callback function is called by the Ascmanager object. This parameter is optional, but, if set, could be used to give back information to the client.
Return values:
none.
