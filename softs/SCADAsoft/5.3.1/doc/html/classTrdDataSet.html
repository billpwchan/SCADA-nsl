Detailed Description

This class is one of the class of the TRD SCADAsoft component.

This class is a collection of TrdData objects which represents the different values of a specific attribute in the database. The archive to which this TrdDataSet belongs specifies the way the attribute is stored (periodically or change-based). This class inherits from the OlsList class.
