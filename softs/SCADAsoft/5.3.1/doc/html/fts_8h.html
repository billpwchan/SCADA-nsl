typedef void(* FtsReturnCallback)(ScsAny arg, ScsStatus returnStatus)

Prototype of the file transfer return callback function. 

 #include "fts.h"
Description:
this function is a function of the client application called by the FTS server to notify the client asynchronously of the result of a file transfer. This callback function is called only once to notify of the result of a file transfer.
Parameters:
arg[in]: Argument supplied in the transfer request.
returnStatus[in]: result status of the file transfer; it may be:
  • ScsValid : success.
  • FtsFtpError : Ftp communication error.
Return values:
none.

Enumeration Type Documentation

enum FtsTransferType

Enum corresponding to the possible file transfer types given back to the return callback function. 

   enum FtsTransferType
   {
     TO_HOST = 0,
     FROM_HOST,
     TO_ENV,
     FROM_ENV
   };
Values:
  • TO_HOST: file transfer to a remote host,
  • FROM_HOST: file transfer from a remote host,
  • TO_ENV: file transfer to a remote SCADAsoft environment,
  • FROM_ENV: file transfer from a remote SCADAsoft environment.
