enum ScsTraceMode 
 #include "scs.h"
Description:
enum represents the type of trace output (see Scadasoft::SetTraceOutput()). The possible values are the following : 
   enum ScsTraceMode {
     ScsCOUT = 0,
     ScsCLOG = 1
   };
Values:
  • ScsCOUT: the traces are redirected to the standard output,
  • ScsLOG: the traces are redirected to a log file.
