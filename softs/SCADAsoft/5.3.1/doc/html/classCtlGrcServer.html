Detailed Description

CTL Grouped Remote Control Server structure for the CTLGRC component.

This class is used to describe a command to send to an equipment. A command corresponds to an internal output variable (IOV). 

 #include "ctlgrc.h"

typedef void(* CtlGrcServer::CtlGrcCompletionCallback)(const char *environment, const char *grcName, CtlGrc::ScsCtlGrcUpdateType type, long completionStatus, long stepNumber, ScsAny callbackArg)

Prototype of a GRC execution callback function. 

 #include "ctlgrc.h"
Description:
this callback is called each time either the current GRC step, and the current GRC status change.
Parameters:
environment[in]: logical environment name where the GRC is running.
grcName[in]: GRC name.
type[in]: type of updating (Brc or Grc).
completionStatus[in]: current GRC execution status.
stepNumber[in]: current execution step.
callbackArg[in]: user optional argument.
Return values:
none.
