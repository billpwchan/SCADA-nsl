Detailed Description

This class is used to store the address of a database point attribute, and, eventually, its value and its type.

The DbmData class is used to store address and value of a database point attribute. The point address is a symbolic description and is stored in an ScsAddress object. The value is stored in an ScsData object with its associated type. 

 #include "dbm.h"
