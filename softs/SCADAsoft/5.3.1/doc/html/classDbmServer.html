Detailed Description

This class is one of the interface classes for the exported services of the DBM component (DataBase Manager): it allows to access to the database for reading/writing or configuring.

This class is the interface class for the services exported by the DbmServer. It is used to read/write sets of data values into database, to interrogate the database for configuration information, and to create or delete points in database.

Member Enumeration Documentation

enum DbmServer::AttributeCategory
Description:
enum corresponding to the possible values for the category of an attribute (see addAttribute()). 
     enum AttributeCategory { 
         Static = 0,
         Dynamic,
         HistoricST,
         HistoricMT
     };
Values:
  • Static: the value of a static attribute is stored only in a real-time context,
  • Dynamic: the value of a dynamic attribute is stored in several context including the real-time context,
  • HistoricST: the value of a historic short-term attribute is stored like a volatile attribute and is stored every short-term period in a short-term historisation buffer,
  • HistoricMT: the value of a historic medium-term attribute is stored like historic the short-term attribute and is also stored every medium-term period in a medium-term historisation buffer.
enum DbmServer::CEOperationMode
Description:
enum corresponding to the possible values of operation modes for CE control (see controlCE() or getCEOper()). 
     enum CEOperationMode { 
         SCS_CE_DISABLED = 0,
         SCS_CE_DISABLED_NORUN,
         SCS_CE_ENABLED,
         SCS_CE_ENABLED_OPT_A,
         SCS_CE_ENABLED_NORUN
     };
Values:
  • SCS_CE_DISABLED: disable the CE, running it one more time to update attribute qualities,
  • SCS_CE_DISABLED_NORUN: disable the CE, without running it,
  • SCS_CE_ENABLED: enable the CE, running it immediately (target point and all dependencies triggered on every write),
  • SCS_CE_ENABLED_OPT_A: enable the CE, running it immediately. DB write triggers target point only if other attributes in the target depend on it. All other points which depend on the attribute written are triggered.
  • SCS_CE_ENABLED_NORUN: enable CE, without running it.
