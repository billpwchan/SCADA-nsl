typedef void(* DacUpdateCallback)(unsigned int subscriptionId, unsigned int eivCount, DacEIV *eivList, ScsAny usrArg)

Prototype of the external input variable (EIV) notification callback function. 

 #include "dac.h"
Description:
This function is a function of the client application for external input variables notification. A client can subscribe to the DACCONTROL server to a list of external input variables, and the server will call this callback function to asynchronously notify the client about changes in values or statuses of the external input variables the client subscribed to.
Parameters:
subscriptionId[in]: identifier of the subscription.
eivCount[in]: number of external input variables (EIV).
eivList[in]: list of external input variables.
usrArg[in]: user optional argument (not used by DAC component). This argument is given by the client when it calls the subscription function and returned to it when the current callback function is called by the DAC component. This parameter is optional, but, if set, could be used to give back information to the client.
Return values:
none.
